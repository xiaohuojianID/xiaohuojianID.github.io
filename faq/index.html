📱 账号相关

关于小火箭账号类型、使用方式、安全性的常见疑问。

小火箭账号指的是用于下载 Shadowrocket（小火箭）的 Apple ID。Shadowrocket 仅在 iOS App Store 上架，由于国区 App Store 下架了所有代理类 VPN 应用，用户需要通过外区 Apple ID（如美区、港区、日区等）才能搜索和下载。这些专门用于下载 Shadowrocket 的外区 Apple ID，被广泛称为「小火箭账号」。

共享账号是多人共用的 Apple ID，仅供登录 App Store 下载 Shadowrocket 使用。用户不需要登录 iCloud，下载完成后退出账号即可。共享账号价格低廉，适合一次性下载、偶尔使用、预算有限的用户。正规渠道的共享账号会控制同时使用的人数，避免触发 Apple 的安全风控机制。

独享账号是专属于你一个人的 Apple ID，没有其他人共用。你可以自由修改密码、密保问题、安全设置，甚至可以登录 iCloud 同步数据。独享账号安全性最高，不会被他人锁定或滥用，适合长期使用、重视安全、需要在多台设备间同步的用户。

共享账号多人共用，只能登录 App Store 下载，不能改密码和保密资料，价格低；独享账号一人独享，可以改密码、登录 iCloud，安全性和稳定性更高，价格也相对高一些。如果你只是下载一次 Shadowrocket，共享账号已经够用；如果长期使用且注重安全，建议选独享账号。

共享账号不能修改密码，因为密码是共用的，改了其他人就无法使用了。独享账号可以自由修改密码和密保资料，购买后建议第一时间修改密码，确保只有自己能使用。

共享账号的邮箱是固定的，不能修改。独享账号的邮箱一般也是固定的（由注册时的邮箱决定），部分独享账号支持修改。购买前建议先确认清楚。

Apple ID 本身没有严格的设备数量限制，可以在多台 iPhone、iPad 上登录 App Store 下载。但共享账号会控制同时在线人数，避免触发风控。独享账号则完全由你自己使用，想登几台设备都行。

共享账号本质上是多人共用的资源。如果一个人修改了密码或密保资料，其他所有使用该账号的人都无法登录了。为了保障所有人的正常使用，共享账号的登录信息是固定的，不开放修改权限。

正规渠道的共享账号定期更换密码，以防止滥用和锁定。如果你购买的是长期共享账号，服务商会持续维护，不会出现「掉号」问题。独享账号完全由你掌控，只要你保管好密码就不会掉。

Apple ID 本身没有有效期限制，只要你不违规使用，账号可以一直存在。独享账号一旦购买归你所有，可以一直使用。共享账号只要服务商在维护，也可以长期使用。

共享账号通常是自动发货，支付后秒到。独享账号需要手动创建或分配，一般在 5-30 分钟内发货。如果长时间未收到账号信息，请联系客服处理。

收到账号后在 App Store 登录（不要登录设置里的 iCloud），搜索下载 Shadowrocket。下载完成后立即退出该账号，换回你自己的 Apple ID。之后的更新可以在 App Store 已购项目中找到并更新。

共享账号仅在 App Store 登录，不登录 iCloud。对方无法看到你的通讯录、照片、信息等个人数据。你的个人信息是安全的，下载完成后退出账号即可完全断开关联。

🚀 使用

关于 Shadowrocket 使用技巧、连接问题、速度优化等。

选择一个节点 → 打开顶部的连接开关 → 首次连接会弹出 VPN 配置授权提示，点击「允许」并输入锁屏密码 → 连接成功后状态栏会出现 VPN 图标。再次点击连接开关即可断开。

确认节点在线、服务器正常。尝试切换为全局路由模式。更换不同节点试试。重启 VPN 连接（关闭再打开）。如果仍然失败，检查是否有其他 VPN 应用冲突，或重启手机再试。

速度慢的常见原因：节点本身带宽有限、国际线路晚高峰拥堵、本地网络问题。尝试切换到其他节点，使用有线网络或更好的 Wi-Fi，避开高峰时段使用。如果长期速度慢，建议考虑更换服务商。

切换到全局模式测试能否访问。如果全局模式能打开但规则模式打不开，说明是规则配置的问题。检查 DNS 设置是否正常，尝试切换「配置」为默认配置。也有可能是节点本身无法访问目标网站。

DNS 污染是指 DNS 查询被中间设备拦截并返回错误的 IP 地址，导致网站无法访问。Shadowrocket 内置了 DNS 防污染功能，可以在设置中开启「加密 DNS」或使用 DoH/DoT 来解决。

在 Shadowrocket 主页面的节点列表中，点击任意节点即可切换。切换后连接会自动断开并重新连接到新节点。建议切换前先测速，选择延迟较低的节点。

全局模式下所有流量都经过代理服务器；配置（规则）模式根据预设规则智能分流，国内网站直连、国外网站走代理。日常使用推荐配置模式，兼顾速度和可用性。测试节点时可以用全局模式。

Shadowrocket 支持自定义规则。进入配置页面 → 选择当前配置 → 可以编辑规则列表。可以添加域名、IP 段、GeoIP 等规则，指定其走代理或直连。也可以直接导入现成的规则配置文件。

场景模式允许你根据不同 Wi-Fi 网络自动切换配置。打开 Shadowrocket → 主页底部「场景」→ 添加新场景 → 选择触发条件（特定 Wi-Fi / 时间 / 位置）→ 绑定对应的配置和节点。适合在不同网络环境间自动切换。

分流是 Shadowrocket 的核心功能。通过预设规则，让国内网站直连、国际网站走代理，既不影响国内访问速度，又能正常使用国际服务。可以通过导入第三方规则配置来优化分流效果。

Shadowrocket 支持 IPv6。在设置中开启「IPv6」支持即可。注意：如果你的网络本身不支持 IPv6，开启后可能会导致部分应用连接异常。建议仅在需要时开启。

Shadowrocket 支持的协议包括：Shadowsocks、ShadowsocksR、V2Ray（VMess）、Trojan、HTTP、HTTPS、Socks5、TUIC、Hysteria2、Sing-box 等。几乎覆盖了当前主流的所有代理协议。

Shadowrocket 支持开机自动连接。打开 Shadowrocket → 设置 → 开启「连接时启动」或「自动连接」。也可以利用 iOS 的快捷指令（Shortcuts）自动化功能，设置连接特定 Wi-Fi 时自动开启 VPN。

Shadowrocket 在正常使用时耗电量很低，对续航影响微乎其微。但如果长时间保持连接并传输大量数据，耗电会相应增加。不需要时建议关闭 VPN 连接以节省电量。

iOS 系统可能会在内存不足时杀掉后台应用，但 Shadowrocket 作为 VPN 应用享有系统级后台权限，通常不会被杀掉。如果发现频繁断连，检查是否开启了省电模式，或确认设置中的后台刷新权限已开启。

在 App Store 中登录下载时使用的 Apple ID → 进入更新页面 → 找到 Shadowrocket → 点击更新。如果国区已下架，需要用原来下载时使用的外区 Apple ID 登录才能看到更新。

Shadowrocket 同一时间只能连接一个节点，不支持多节点同时连接。但你可以通过规则配置实现分流：不同类型的流量走不同的策略。例如：观看 Netflix 走香港节点，访问 Google 走美国节点。

Shadowrocket 支持基于规则的广告屏蔽。导入包含广告拦截规则的配置（如 Anti-AD、AdGuard DNS 规则），即可在代理层面拦截广告请求。可以在规则配置中加入广告域名列表，将它们指向「Reject」策略。

Shadowrocket 设置中有导出配置的功能，可以将配置分享到文件、AirDrop 或 iCloud。建议在修改重要规则前先备份当前的配置文件。也可以利用 iCloud 在不同设备间同步配置。

如果 Shadowrocket 出现异常，可以在设置中点击「重置配置」，恢复所有设置为默认值。也可以在 iPhone 的「设置 → 通用 → VPN 与设备管理」中删除 Shadowrocket 的 VPN 配置，然后重新打开应用重新配置。
