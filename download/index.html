--- layout: page title: 下载中心 description: Shadowrocket 小火箭下载指南 — iOS App Store 下载、外区 Apple ID 注册、安装教程与常见错误解决。 show_cta: false --- ## iOS 下载 Shadowrocket 仅在 iOS App Store 上架，由于国区 App Store 已下架代理类应用，你需要使用**外区 Apple ID**（如美区、港区）才能下载。 ### 下载步骤 1. **获取外区 Apple ID**：注册一个美区或其他外区的 Apple ID，或使用共享账号 2. **登录 App Store**：打开 App Store，点击右上角头像，退出当前账号 3. **输入外区 Apple ID**：登录后，App Store 会自动切换到对应区域 4. **搜索 Shadowrocket**：在 App Store 搜索框输入「Shadowrocket」 5. **下载安装**：点击获取/安装按钮，等待下载完成 6. **退出外区账号**：下载完成后，务必退出外区 Apple ID，换回自己的账号 > ⚠️ **重要提醒**：下载完成后一定要退出外区 Apple ID！不要在设置中登录 iCloud，否则可能导致账号被锁或数据丢失。 ## Apple ID 指南 ### 美区 Apple ID 注册 如果你想自己注册一个美区 Apple ID，可以参考以下教程。注册时需要准备一个未绑定 Apple ID 的邮箱。 详细教程请查看 [Apple ID 注册指南](/ios/)。 ## 常见下载错误 ### 无法连接到 App Store - 检查网络连接是否正常 - 尝试切换 Wi-Fi 或使用蜂窝数据 - 确认 App Store 没有被限制访问 ### 提示 "Apple ID 已被锁定" - 共享账号可能因多人同时登录被 Apple 安全机制锁定 - 联系账号提供方获取解锁或更换账号 ### 提示 "此 Apple ID 尚未在 iTunes Store 使用过" - 首次使用外区 Apple ID 时需要同意条款 - 在 App Store 中尝试下载任意免费应用完成激活 ### 如何恢复已购应用 如果你之前已下载过 Shadowrocket 但换了设备： 1. 登录原来下载时使用的 Apple ID 2. 打开 App Store → 右上角头像 → 已购项目 3. 找到 Shadowrocket 点击下载按钮（云朵图标） > 📌 更多错误解决方案请查看 [错误解决中心](/errors/)。
