Clash Meta

推荐 开源免费

跨平台 · TUN 模式 · 规则集 · 高性能核心

Clash Meta（现更名为 Mihomo）是 Clash 生态中最强大的进阶分支，在保留原始 Clash 全部功能的基础上，加入了大量企业级特性。它是目前性能最优、功能最全的 Clash 核心之一，被 Clash Verge、Clash Nyanpasu 等主流 GUI 客户端作为默认内核使用。

核心特性：

  • TUN 模式 — 支持虚拟网卡级代理，无需逐个配置应用，所有流量自动接管
  • 规则集（Rule-Set） — 支持远程规则集、本地规则集、GeoSite 和 GeoIP 数据库，规则管理更灵活
  • DNS 劫持 — 内置 DNS 模块，支持 fake-ip、redir-host 等多种 DNS 解析模式
  • 连接嗅探 — 自动识别 TLS/HTTP/QUIC 等协议流量，智能分流
  • gRPC/HTTPS 订阅 — 支持 gRPC 和 HTTPS 协议的订阅传输，突破网络限制
  • 性能优异 — Go 语言编写，内存占用低，并发处理能力强，适合长期运行

Clash Meta 本身是命令行核心，普通用户建议搭配 Clash Verge 等 GUI 客户端使用。进阶用户可直接通过配置文件 + 命令行方式运行，实现更精细的流量控制。

Clash Verge

开源免费

Windows/macOS/Linux · 现代化 GUI · 订阅管理 · 规则编辑

Clash Verge 是目前最受欢迎的 Clash 图形界面客户端之一，采用 Tauri + React 技术栈构建，界面现代美观，操作直观。它默认集成 Clash Meta 核心，开箱即用，特别适合从零开始的新手用户。

核心特性：

  • 一键订阅管理 — 支持粘贴订阅链接自动导入，定时自动更新，多订阅同时管理
  • 内置规则编辑器 — 可视化编辑 YAML 配置文件，支持语法高亮、代码补全，无需手动编辑
  • TUN 模式开关 — 一键开启/关闭 TUN 模式，切换全局代理与规则模式
  • 流量监控 — 实时显示上下行速度、总流量消耗、节点延迟
  • 系统代理管理 — 自动配置系统代理，支持 PAC 模式、全局模式、规则模式三种运行方式
  • 多语言支持 — 内置简体中文、繁体中文、英文等多语言界面
  • 便携版可选 — 提供免安装便携版，U盘即可随身携带

Clash Verge 是目前新手入门的最佳选择。它降低了 Clash 的使用门槛，同时保留了高级配置入口，随着使用经验的增长可以逐步探索更多高级功能。推荐所有初次接触 Clash 的用户优先选择。

Sing-box

开源免费 全平台

通用代理平台 · Go 编写 · 轻量高性能 · Clash 替代方案

Sing-box 是由 Clash 原作者 Dreamacro 的后续团队开发的新一代通用代理平台，使用 Go 语言从零编写。它被设计为 Clash 的现代化替代品，在保持高性能的同时大幅简化了配置复杂度。目前已支持 Windows、macOS、Linux、iOS、Android 全平台。

核心特性：

  • 统一配置格式 — 使用 JSON 配置文件，结构清晰，比 YAML 更易解析和调试
  • 多协议原生支持 — 内置 SS/V2Ray/Trojan/Hysteria/ShadowTLS 等协议支持，无需额外插件
  • TUN 模式 — 全平台 TUN 虚拟网卡支持，可实现全局代理
  • 灵活的路由规则 — 支持域名、IP、GEOIP、GeoSite、端口、协议等多维度路由
  • DNS 模块 — 内置完整 DNS 解析器，支持 DNS-over-HTTPS、DNS-over-TLS、fake-ip
  • 多平台客户端 — 官方提供 iOS 端（Sing-box for iOS）、 Android 端，命令行核心跨平台
  • Clash 配置转换 — 支持将 Clash YAML 配置自动转换为 Sing-box JSON 格式

Sing-box 的 iOS 端是目前 App Store 上最推荐的免费代理客户端之一，功能不输付费软件。命令行核心非常适合服务器和嵌入式设备使用。

OpenClash

开源免费

OpenWrt · 路由器全局代理 · 多核心支持

OpenClash 是 OpenWrt 路由器上最强大的 Clash 插件，可以让您的路由器实现全局科学上网——连接路由器的所有设备自动享有代理服务，无需逐个配置。支持旁路由模式，适合软路由和硬路由用户。

核心特性：

  • 多核心支持 — 支持 Clash、Clash Meta、Clash Premium、Sing-box 等多种核心，可自由切换
  • TUN/Redir/Fake-IP 三种模式 — 根据路由器性能和应用场景选择最合适的运行模式
  • 订阅管理 — 支持订阅链接导入、自动更新、多订阅合并
  • 规则集管理 — 支持 ACL4SSR、Loyalsoldier 等第三方规则集，自动更新规则
  • 仪表盘 — Web 管理面板，实时查看连接状态、节点延迟、流量统计
  • 旁路由支持 — 作为旁路由运行时不影响主路由稳定性，适合尝鲜用户

OpenClash 适合有一定路由器操作经验的用户。安装前请确保 OpenWrt 固件留有足够存储空间（建议 256MB+），推荐使用 Lean 或 ImmortalWrt 等常用固件。
