--- layout: page title: 共享账号 description: 了解小火箭共享账号的使用方法、注意事项和常见问题。安全使用共享 Apple ID 下载 Shadowrocket。 --- ## 什么是共享账号？ 共享账号是一种多人共用的 Apple ID，主要用于在 App Store 下载 Shadowrocket 等应用。用户只需使用该账号登录 App Store（而非 iCloud），下载完成后退出即可。 ## 共享账号的使用方法 ### 第一步：登录 App Store 1. 打开 iPhone/iPad 的 **App Store** 2. 点击右上角的头像 3. 滚动到底部，点击「退出登录」 4. 输入共享账号的 Apple ID 和密码 5. 点击「登录」 > ⚠️ **重要**：只在 App Store 登录！不要打开「设置」→ Apple ID 进行登录！ ### 第二步：下载应用 1. 登录后在 App Store 搜索「Shadowrocket」 2. 点击获取/安装按钮 3. 等待下载完成 ### 第三步：退出账号 1. 下载完成后，再次点击 App Store 右上角头像 2. 滚动到底部，点击「退出登录」 3. 重新登录你自己的 Apple ID ## 使用注意事项 {% include tip-box.html type="warning" title="绝对不要做的事" %} - ❌ **不要在「设置」中登录**：只在 App Store 使用 - ❌ **不要登录 iCloud**：会导致数据同步混乱 - ❌ **不要绑定支付方式**：不要添加信用卡或支付宝 - ❌ **不要开启双重认证**：会影响其他人使用 - ❌ **不要修改账号信息**：密码、安全问题等不要改动 {% include tip-box.html type="tip" title="安全使用建议" %} - ✅ 下载完成后立即退出 - ✅ 从正规渠道获取共享账号 - ✅ 定期更换使用的共享账号 - ✅ 重要数据备份在自己的 Apple ID 上 ## 常见问题 ### 登录时提示「Apple ID 已锁定」？ 账号可能因多次登录失败或被 Apple 判定异常而锁定。请联系账号提供方解锁或更换账号。 ### 下载按钮是灰色的？ 可能是 App Store 区域未正确切换。确认登录后 App Store 页面语言是否变为英文。如果仍是中文，退出重新登录即可。 ### 可以同时在多台设备登录吗？ 不建议。多台设备同时登录同一共享账号可能触发 Apple 安全机制，导致账号被锁定。
