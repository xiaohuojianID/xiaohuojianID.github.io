--- layout: page title: 错误解决中心 description: Shadowrocket 常见错误与解决方案 — 下载失败、安装错误、配置问题、连接异常等。 show_cta: false --- ## 下载类错误 ### 无法连接到 App Store **症状**：打开 App Store 后显示「无法连接」或白屏。 **解决方案**： 1. 检查网络连接（Wi-Fi / 蜂窝数据） 2. 尝试开启飞行模式再关闭 3. 重启设备 4. 检查「设置 → 屏幕使用时间 → 内容和隐私访问限制」是否限制了 App Store ### 提示「Apple ID 已被锁定」 **症状**：登录时提示「您的 Apple ID 已被锁定」。 **解决方案**： - 共享账号：联系提供方解锁或更换 - 个人账号：访问 [iforgot.apple.com](https://iforgot.apple.com) 解锁 ### 提示「此 Apple ID 尚未在 iTunes Store 使用过」 **症状**：首次使用外区 Apple ID 时弹出提示。 **解决方案**：在 App Store 中先下载任意免费应用（如 Twitter、WhatsApp），同意条款后即可正常使用。 --- ## 安装类错误 ### 下载后无法安装 **可能原因与解决**： - 存储空间不足 → 清理空间后重试 - iOS 版本过低 → 升级到 iOS 12.0 以上 - 网络不稳定 → 切换网络重试 ### 安装后找不到应用 **解决方案**：在主屏幕右滑到最后一页打开 App 资源库，搜索「Shadowrocket」。 --- ## 配置类错误 ### 订阅链接导入失败 **排查步骤**： 1. 检查链接格式是否正确 2. 用 Safari 打开链接验证是否可访问 3. 尝试使用蜂窝数据而非 Wi-Fi 4. 检查链接是否已过期或被禁用 ### 节点全部超时 **可能原因**： - 当前网络环境限制了代理连接 - 服务器端故障 - 本地 DNS 问题 **解决**： 1. 更换网络环境尝试 2. 切换不同节点 3. 在 Shadowrocket 设置中切换 DNS --- ## 连接类错误 ### 连接后无法上网 1. 切换路由模式为「全局路由」 2. 检查节点是否真实在线 3. 尝试不同节点 4. 关闭其他 VPN 应用 ### VPN 频繁断开 1. 关闭省电模式 2. 开启 Shadowrocket 后台刷新 3. 更换更稳定的节点 4. 检查是否被其他 VPN 应用冲突 ### 提示 VPN 配置失败 1. 删除旧的 VPN 配置：「设置 → 通用 → VPN 与设备管理 → 删除 Shadowrocket 配置」 2. 重新打开 Shadowrocket 添加配置 3. 如仍失败，重启设备后重试 --- ## 其他常见问题 ### Shadowrocket 无法更新 使用下载时所用的 Apple ID 登录 App Store 即可更新。国区已下架，不能用国区账号更新。 ### 恢复购买失败 确认你登录的 Apple ID 与购买时使用的 Apple ID 一致。Shadowrocket 是买断制，不会重复收费。
