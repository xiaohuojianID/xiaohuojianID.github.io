⬇️ 下载教程

### Shadowrocket 怎么下载？ Shadowrocket 是 iOS 平台最知名的网络代理工具之一，仅上架 Apple App Store。由于国区 App Store 下架了所有代理类 VPN 应用，你需要使用外区 Apple ID（美区、港区、日区等）才能搜索和下载。 **下载步骤：** 1. **准备一个外区 Apple ID** — 可以在我们这里购买小火箭共享账号或独享账号 2. **打开 App Store**，点击右上角头像 3. **退出当前 Apple ID**，滑到底部点击「退出登录」 4. **登录外区 Apple ID**，输入账号密码 5. **搜索「Shadowrocket」** — 注意拼写正确 6. **点击下载** — 售价 $2.99，绑定该 Apple ID 7. **下载完成后退出外区账号**，换回自己的 Apple ID > 💡 **提示**：下载完成后 Shadowrocket 就属于你的 Apple ID 了。以后换手机或重装系统，登录原来的 Apple ID 在已购项目中可以重新下载，不需要再次付费。 ### App Store 下载 登录外区 Apple ID 后，在 App Store 搜索框输入「Shadowrocket」。如果搜不到，检查以下几点： - 确认当前 App Store 显示的是英文界面（说明已经切换到美区） - 如果还是中文界面，说明还是国区，退出重新登录外区账号 - 尝试搜索「Shadowrocket」完整名称，不要打错字母 - 重启 App Store 应用后再搜索 ### Apple ID 登录 登录时注意： - **只在 App Store 登录**，不要登录「设置」里的 iCloud - 如果你用的是共享账号，下载完成后立即退出 - 不要在这个账号上绑定你自己的支付方式或手机号 - 如果提示需要验证，联系客服处理，不要多次尝试错误密码 ### 恢复购买 换了新 iPhone 或重装系统后： 1. 登录当初下载时使用的 Apple ID 2. 打开 App Store → 右上角头像 → 已购项目 3. 找到 Shadowrocket，点击旁边的云朵图标 4. 下载完成后退出该账号 ---

📲 安装教程

### 安装 Shadowrocket 下载完成后，Shadowrocket 会自动安装到你的 iPhone 或 iPad 桌面。如果桌面找不到： 1. 在主屏幕向右滑动，打开 **App 资源库** 2. 在搜索框输入「Shadowrocket」 3. 长按图标，选择「添加到主屏幕」 ### 首次配置 第一次打开 Shadowrocket，你会看到： 1. **VPN 配置授权** — 点击「允许」 2. **输入锁屏密码** — 确认授权 3. **主界面** — 你会看到节点列表、连接开关和底部菜单 ### 允许 VPN Shadowrocket 通过 iOS 的 VPN 机制工作，首次连接时系统会提示添加 VPN 配置。点击「允许」并输入你的 iPhone 锁屏密码。这是正常的安全验证，所有 VPN 应用都需要这一步。 ### 添加订阅 这是使用 Shadowrocket 最重要的一步： 1. 点击右上角 **+** 2. 类型选择 **Subscribe**（订阅） 3. 在 URL 栏粘贴你的订阅链接 4. 点击「完成」 5. 保存后自动拉取节点列表 节点列表会显示在主页面上，每个节点右侧有延迟测试结果。 ---

⚙️ 配置教程

### 导入订阅 订阅是获取代理节点最方便的方式。你从机场服务商那里购买的订阅链接，可以直接导入 Shadowrocket。 **操作步骤：** 1. 复制订阅链接（通常由服务商提供） 2. 打开 Shadowrocket → 右上角 **+** 3. 类型选择 **Subscribe** 4. 粘贴链接 → 点击「完成」 5. 页面会自动拉取所有节点 ### 更新订阅 节点列表需要定期更新才能保持最新： - **手动更新**：主页找到订阅条目 → 向右滑动 → 点击「更新」 - **批量更新**：主页下拉刷新，所有订阅同时更新 - **自动更新**：设置 → 订阅 → 开启「自动更新」，设置更新间隔 ### 节点切换 在主页面的节点列表中，点击你想使用的节点即可切换连接。建议选择延迟较低的节点，方法： 1. 点击顶部的 **⚡ 测速按钮**，对所有节点进行延迟测试 2. 绿色延迟表示响应快，黄色次之，红色最慢 3. 点击选择延迟低的节点 ### 规则模式 Shadowrocket 支持三种路由模式： - **全局路由**：所有流量都经过代理，适合需要完全代理的场景 - **配置（规则）**：根据预设规则智能分流，国内网站直连、国外网站走代理（推荐日常使用） - **场景模式**：根据不同 Wi-Fi 或时间自动切换配置 日常使用推荐「配置」模式，兼顾国内访问速度和国际连接。 ---

🚀 高级教程

### DNS 设置 DNS 解析直接影响访问速度和稳定性。Shadowrocket 支持多种 DNS 配置： - **加密 DNS（DoH/DoT）**：防止 DNS 污染和劫持 - **自定义 DNS**：可以指定使用 Cloudflare（1.1.1.1）、Google（8.8.8.8）等公共 DNS - **DNS 分流**：国内域名走国内 DNS，海外域名走加密 DNS **设置路径**：Shadowrocket → 设置 → DNS ### 分流策略 分流是 Shadowrocket 最强大的功能之一。通过预设的规则配置，可以实现： - 国内网站直接连接，不影响速度 - 国际网站通过代理访问 - 特定应用或域名走指定线路 - 广告域名直接拦截 推荐使用成熟的规则配置（如 ACL4SSR、Loyalsoldier 等），可以直接导入使用。 ### 规则配置 进入「配置」页面，可以查看和编辑当前使用的规则文件。每条规则包含： - **类型**：DOMAIN（域名）、IP-CIDR（IP 段）、GEOIP（地区 IP）等 - **策略**：Proxy（代理）、Direct（直连）、Reject（拦截） - **匹配规则**：支持通配符和正则表达式 你可以导入社区维护的规则集，也可以根据自己的需求手动添加规则。 ### 代理模式 Shadowrocket 支持多种代理模式： - **Shadowsocks**：轻量级加密代理，速度快 - **V2Ray**：功能丰富的代理协议，支持多种传输方式 - **Trojan**：基于 HTTPS 的代理协议，伪装性好 - **TUIC / Hysteria2**：基于 QUIC 的新一代协议，弱网环境表现好 不同的协议适合不同的使用场景，选择哪种取决于你的服务商支持和网络环境。 ### IPv6 支持 如果你的网络支持 IPv6，可以在设置中开启 IPv6。注意： - 开启后部分仅支持 IPv4 的应用可能连接异常 - 建议仅在需要访问 IPv6 资源时开启 - 如果遇到连接问题，先关闭 IPv6 测试 ---

🔧 问题解决

### 连接失败 **常见原因和解决方法：** 1. **节点不可用** — 尝试切换到其他节点 2. **网络问题** — 切换 Wi-Fi 和蜂窝数据试试 3. **VPN 冲突** — 关闭其他 VPN 应用 4. **配置错误** — 检查订阅链接是否有效 如果以上都不行，尝试重启 Shadowrocket 或重启手机。 ### 下载失败 App Store 下载失败的处理： 1. 检查 iPhone 存储空间是否足够 2. 切换网络环境（Wi-Fi 换蜂窝数据） 3. 登录正确的 Apple ID 4. 重启 App Store 应用 5. 重启手机后重新下载 ### 闪退 Shadowrocket 闪退的常见原因： 1. **iOS 版本过低** — 需要 iOS 12.0 以上 2. **内存不足** — 关闭后台其他应用 3. **配置冲突** — 重置配置后重试 4. **应用损坏** — 删除后重新下载 ### 超时 节点连接超时处理： 1. 切换到其他节点测试 2. 切换到全局路由模式 3. 更换网络环境（Wi-Fi 换 4G/5G） 4. 使用不同协议测试 5. 如果所有节点都超时，可能是你的本地网络问题 ### 更新失败 订阅更新失败处理： 1. 检查订阅链接是否过期 2. 用 Safari 打开链接验证可访问 3. 切换到蜂窝数据更新 4. 删除订阅后重新导入 5. 联系服务商确认订阅状态 ---
