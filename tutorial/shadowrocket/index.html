Shadowrocket（俗称"小火箭"）是 iOS 平台上最受欢迎的代理客户端工具，支持 Shadowsocks、SSR、V2Ray、Trojan、VLESS 等多种代理协议。它界面简洁、功能强大，是 iPhone 和 iPad 用户实现网络代理的首选工具。

本教程将从购买安装开始，一步步带你完成 Shadowrocket 的完整配置流程。无论你是第一次使用 iOS 代理的新用户，还是希望深入了解 Shadowrocket 进阶玩法（如规则编辑、按需连接、快捷指令等）的老用户，本教程都值得收藏。

1

购买并安装 Shadowrocket

Shadowrocket 是一款付费软件，售价 $2.99（约人民币 18-22 元），需使用美区或其他支持 Shadowrocket 的 Apple ID 购买。

方法一：美区 Apple ID

  1. 注册一个美区 Apple ID（或购买现成的）
  2. 在 App Store 中登录美区账号
  3. 搜索 "Shadowrocket" 并购买下载
  4. 下载后切换回国区账号也可正常使用

方法二：共享/兑换码

  1. 通过某些渠道获取共享账号或兑换码
  2. 在 App Store 中登录兑换
  3. ⚠️ 注意共享账号存在安全隐患
  4. 建议自己购买，长期使用更安心

📌 Shadowrocket 官方 App Store 链接：apps.apple.com/app/shadowrocket/id932747118

2

导入订阅链接

安装完成后，打开 Shadowrocket，按照以下步骤导入你的机场订阅：

  1. 1

    打开 Shadowrocket，点击底部导航栏的 配置 选项卡

    启动后主界面底部有"首页"、"配置"、"数据"、"设置"四个标签页

  2. 2

    点击右上角的 + 号按钮

    弹出"添加节点"和"添加订阅"选项

  3. 3

    选择 添加订阅，在 URL 输入框中粘贴你的订阅链接

    可自定义备注名称方便识别

  4. 4

    点击 保存，Shadowrocket 将自动下载并解析订阅

    导入成功后返回首页即可看到所有节点

💡 提示

如果你的订阅链接是 Clash 格式（YAML），但 Shadowrocket 无法直接识别，可以先使用在线订阅转换工具（如 subconverter）将订阅转换为 Shadowrocket 支持的格式。

3

节点选择与测速

订阅导入成功后，返回 Shadowrocket 首页即可看到所有节点。每个节点左侧有信号图标和延迟信息。

测速方法

  • • 点击首页右上角的 测速 按钮
  • • 选择测速模式（连接测试 / 速度测试）
  • • 连接测试：快速检测延迟（推荐）
  • • 速度测试：检测下载速度（耗流量）
  • • 测试完成后节点将按延迟排序

选择建议

  • • 日常浏览：选延迟最低的节点
  • • 看视频：选速度快的节点
  • • 玩游戏：选延迟稳定、丢包低的
  • • 解锁流媒体：选对应地区的节点

📌 点击节点右侧的 感叹号图标 可以查看节点的详细信息，包括协议类型、加密方式、服务器地址等。通过 长按 节点可将其设为默认节点。

4

开启代理

选择好节点后，只需一步即可开启代理。Shadowrocket 提供了两种启动方式：

方式一：首页开关

在 Shadowrocket 首页，点击顶部的连接开关按钮。开启后状态栏会显示 VPN 图标，表示代理已生效。

方式二：快捷指令

Shadowrocket 支持通过 iOS 快捷指令一键开关代理。在"快捷指令"App 中添加 Shadowrocket 操作即可设置自动化场景，例如连接特定 Wi-Fi 时自动开启。

💡 提示：开启代理后，你可以在 设置 → 全局路由 中选择代理模式：配置（规则分流）、全局（全部代理）、直连（全部直连）。日常使用建议保持"配置"模式。

5

规则与分组设置

Shadowrocket 的规则系统非常灵活，可以通过编辑配置文件实现精细化的流量分流管理。

规则编辑方法：

1

进入 配置 选项卡 → 点击当前使用的配置文件 → 选择 编辑规则

2

点击右上角的 + 添加新规则，或长按现有规则进行编辑/排序

3

规则格式：类型, 匹配值, 策略，例如：DOMAIN-SUFFIX, google.com, Proxy

节点分组设置：

在配置文件中可以设置策略组，将多个节点放入一个分组中。例如：

[Proxy Group]
Auto = select, url=http://www.gstatic.com/generate_204, interval=600
Proxy = select, 🏁 香港 01, 🏁 香港 02, 🇯🇵 日本 01, 🇸🇬 新加坡 01

📌 分组策略说明：select 为手动选择，url-test 为自动测速选择最优节点，fallback 为主备切换，load-balance 为负载均衡。新手建议使用 select 手动选择即可。

❓ Shadowrocket 常见问题

iOS 平台主流代理客户端对比：

  • Shadowrocket（$2.99）：轻量简洁，上手容易，功能够用，性价比最高
  • Surge（$49.99）：功能最强大，支持脚本、MITM、网关模式，适合专业用户和开发者，价格昂贵
  • Quantumult X（$7.99）：功能丰富，支持脚本和重写，介于 Shadowrocket 和 Surge 之间
  • Stash（$2.99）：基于 Clash 内核的 iOS 客户端，支持 Clash 规则集，适合 Clash 用户

如果你是普通用户（只需要浏览网页、看视频），Shadowrocket 完全够用且性价比最高。如果需要高级功能（如 HTTP 抓包、脚本自动化），可以考虑 Surge 或 Quantumult X。

可能的原因和解决方法：

  • 节点问题：当前节点可能无法访问目标网站，切换到其他节点试试
  • DNS 污染：在 Shadowrocket 的设置中开启"DNS 加密"或使用干净的 DNS（如 1.1.1.1、8.8.8.8）
  • 规则问题：检查配置中的规则是否将该网站错误地设置为直连或拒绝
  • 全局路由模式：尝试将全局路由从"配置"切换为"全局"测试
  • 网络环境：某些网络环境（如公司/学校网络）可能限制了代理使用

更新订阅的方法：

  • 手动更新：进入 配置 选项卡 → 长按订阅名称 → 选择"更新订阅"
  • 自动更新：设置 → 订阅 → 开启"自动更新"，并设置更新间隔（建议每 24 小时）
  • 启动时更新：设置 → 订阅 → 开启"启动时更新订阅"

建议开启自动更新功能。如果节点突然全部不可用，首先尝试更新订阅，可能是机场更新了节点信息。

Shadowrocket 的耗电量和影响：

  • 耗电量：正常使用情况下，Shadowrocket 的耗电量很低，约占总电量的 1-3%，几乎可以忽略
  • 网络延迟：开启代理后网络延迟会略有增加（通常 5-30ms），日常使用基本无感
  • 网速影响：取决于节点质量，好节点反而可能加速，差节点会拖慢网速
  • 后台运行：Shadowrocket 会在后台保持 VPN 连接，但 iOS 系统优化得很好，不会显著影响续航
  • 按需连接：在设置中开启"按需连接"，手机会在连接特定网络时自动开启/关闭代理，减少不必要的耗电

总的来说，Shadowrocket 对日常使用的影响非常小，建议保持常开状态。如果你比较在意耗电，可以开启"按需连接"功能。

常见问题及解决方法：

  • 无法安装：确认 App Store 账号所在地支持 Shadowrocket（美区、港区、日区等都可），或切换账号重新登录
  • 闪退：更新 iOS 系统到最新版本，或卸载重装 Shadowrocket
  • 连接失败：检查订阅是否过期 → 尝试更换节点 → 切换全局路由模式 → 重启 App → 重启手机
  • DNS 错误：设置 → DNS → 改为 1.1.1.1 或 8.8.8.8，或开启 DNS 加密
  • VPN 冲突：Shadowrocket 使用 VPN 配置实现代理，请先关闭其他 VPN 类 App
  • 配置重置：在设置 → 最底部 → 点击"重置配置"恢复出厂设置

如果以上都无法解决，建议去机场的售后群或 Shadowrocket 的 Telegram 群组寻求帮助。
